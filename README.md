**Projeto Jogadores**

O Projeto Jogadores é um notebook desenvolvido em Python com o objetivo de analisar dados de jogadores a partir de um arquivo CSV. O projeto realiza a leitura, tratamento e exploração dos dados, gerando gráficos e estatísticas que ajudam a compreender melhor o desempenho, as características e os padrões entre os jogadores.

A proposta é oferecer uma visão clara e interativa dos dados, facilitando a interpretação de métricas como médias, distribuições e correlações. O projeto é ideal para quem deseja aprender ou praticar análise de dados com Python, especialmente utilizando bibliotecas como Pandas e Matplotlib.


**Tecnologias Utilizadas**

Python 3

Jupyter Notebook / Google Colab

Pandas — manipulação e análise de dados

Matplotlib  — geração de gráficos e visualizações


**Como Executar no Google Colab**

Acesse o Google Colab

Clique em “Carregar notebook” e envie o arquivo ProjetoJogadores.ipynb.

No menu lateral esquerdo do Colab, clique na pasta e envie também o arquivo dados_jogadores.csv.

Verifique se o arquivo CSV foi carregado corretamente no mesmo diretório do notebook.

Execute as células do notebook na ordem para realizar a análise.



**Ao final da execução, o notebook gera:**

Estatísticas descritivas sobre os jogadores;

Gráficos de distribuição e correlação entre variáveis;

Conclusões baseadas nos dados analisados.
